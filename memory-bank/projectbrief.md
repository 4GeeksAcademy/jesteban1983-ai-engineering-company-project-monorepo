# Project Brief: TrackFlow

**Objetivo:** Desarrollar la plataforma de gestión logística "TrackFlow", que permite el seguimiento de inventario y optimización de envíos.
**Partes:** 1. Website (Público): Visualización de catálogo y estado de productos.
2. Backoffice (Interno): Gestión de almacenes, transportistas e inventario en tiempo real.
**Problema:** Centralizar la lógica de negocio para evitar duplicidad entre el frontend público y la gestión interna.
